# C2 - Docker Hub 探索題

目標：找一個課堂沒有用過的 Image，閱讀官方文件後，把它跑起來。

建議流程：

1. 到 Docker Hub 搜尋一個你感興趣的 Image。
2. 確認是不是官方或可信任來源。
3. 閱讀：
   - How to use this image
   - Environment Variables
   - Where to Store Data
   - Ports
4. 寫出 `docker run` 指令。
5. 執行並截圖或紀錄結果。
6. 用 `record-template.md` 紀錄你的過程。

範例方向：

- `httpd`
- `postgres`
- `mongo`
- `adminer`
- `wordpress`

不要選本堂已經大量使用的 `nginx`、`mysql`、`redis`。
